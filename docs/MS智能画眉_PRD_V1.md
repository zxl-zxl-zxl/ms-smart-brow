# MS 智能画眉 V1 PRD

## 给 Figma AI 的设计生成提示

请基于以下 PRD 生成微信小程序高保真原型。产品名称为「MS 智能画眉」，副标题为「AI 相机画眉助手」。整体视觉风格为「专业美妆工具感 + 轻科技 AI 感」，使用米白、暖肤色、茶棕、香槟浅渐变作为主视觉方向。首页使用抽象线稿脸部图，不使用真人图；画眉辅助线使用茶棕色细虚线和小圆点，仅描边不填充。请优先设计以下页面：首页、相机定标页、画眉辅助页、微调面板、相机权限引导页、保存成功页、个人中心、隐私说明页、关于我们页。

## 1. 产品定位

产品名称：MS 智能画眉

副标题：AI 相机画眉助手

产品定位：

MS 智能画眉是一款微信小程序画眉辅助工具，通过 AI/智能脸型分析、固定辅助线和实时相机画面，帮助用户找准眉头、眉峰、眉尾，并根据推荐眉形完成画眉。

V1 核心模式：

拍照定标 + 固定辅助层 + 实时相机对齐画眉。

用户点击「开始定标」后，系统识别当前正脸画面并生成固定辅助线。之后用户看着实时相机画面，移动自己的脸与固定辅助线对齐，再沿着眉形虚线画眉。

## 2. V1 核心目标

- 帮助不会画眉或画眉不稳定的用户找到眉形参考。
- 通过固定辅助线降低实时 AR 抖动风险。
- 建立登录用户身份，为后续用户画像、会员、付费模板预留。
- 不上传用户照片，不保存人脸关键点数据，降低隐私风险。

## 3. 用户流程

1. 用户进入首页。
2. 首页展示产品卖点和三步使用说明。
3. 用户点击「开始智能画眉」。
4. 系统触发微信登录。
5. 登录成功后进入相机页。
6. 用户授权相机权限。
7. 用户正脸面对镜头，点击「开始定标」。
8. 系统显示「正在识别脸型...」和轻扫描动画。
9. 识别成功后生成固定辅助线和推荐眉形。
10. 用户可切换眉形模板。
11. 用户可打开微调面板调整辅助线。
12. 用户对齐辅助线后照着眉形虚线画眉。
13. 用户点击「完成并保存」。
14. 系统保存不带辅助线的净图到相册。
15. 保存成功后进入保存成功页。
16. 用户可选择「重新画眉」或「返回首页」。

## 4. 首页需求

### 4.1 视觉风格

- 专业美妆工具感 + 轻科技 AI 感。
- 使用米白、暖肤色、茶棕、香槟浅渐变。
- 主视觉使用抽象线稿脸部图，不使用真人图。
- 主视觉包含眉形虚线、脸部辅助线、轻扫描元素。

### 4.2 页面内容

- 产品名：MS 智能画眉。
- 副标题：AI 相机画眉助手。
- 核心卖点：AI 推荐适合你的眉形；实时相机辅助画眉。
- 三步说明：
  - 点击开始智能画眉。
  - 正脸定标，生成推荐眉形。
  - 对齐辅助线，照着虚线画眉。
- 主按钮：开始智能画眉。
- 右上角：我的。

### 4.3 登录规则

- 点击「开始智能画眉」时，如果未登录，先登录。
- 点击「我的」时，如果未登录，先登录。
- 登录失败或拒绝授权时 Toast：登录后可使用智能画眉服务。
- V1 不获取手机号。

## 5. 相机页需求

### 5.1 设备方向

- V1 只支持竖屏。

### 5.2 页面布局

- 全屏前置相机。
- 左上角：返回首页按钮。
- 相机页不放个人中心入口。
- 定标前中部轻提示：请正脸面对镜头，保持光线充足。
- 底部主按钮：开始定标。

### 5.3 返回规则

- 未定标时点击返回首页，直接返回。
- 已生成辅助线后点击返回首页，需要二次确认。
- 确认文案：返回首页后当前辅助线将清空，是否返回？

## 6. 定标识别需求

### 6.1 定标方式

- 用户手动点击「开始定标」。
- 仅定标瞬间识别一次人脸。
- 画眉过程中不持续做人脸识别。
- 辅助线不会跟随脸部移动。

### 6.2 识别状态

- 点击「开始定标」后显示「正在识别脸型...」。
- 显示轻量扫描动画。
- 识别成功后不弹 Toast，直接展示辅助线和底部眉形卡片栏。

### 6.3 失败规则

- 未识别人脸时 Toast：未识别人脸，请调整光线或距离后重试。
- 多张人脸时 Toast：请确保画面中只有一张人脸后重试。
- 用户侧脸、倾斜过大、脸部不完整时定标失败。
- 支持基础光线/距离判断，过暗、太近、太远时提示重试。
- V1 不提供手动定点兜底。
- V1 不支持上传照片兜底。

## 7. 辅助线需求

### 7.1 辅助线内容

定标成功后生成以下固定辅助线：

- 基础定位线：脸部中线、双眼水平线、左右眉区域参考框。
- 眉形关键点：眉头、眉峰、眉尾。
- 眉形轮廓线：左右眉上边界、下边界虚线轮廓。
- 脸部/五官虚线：脸部轮廓、鼻梁线、眼睛轮廓。

### 7.2 视觉规范

- 眉形轮廓只显示轮廓线，不填充颜色。
- 眉形轮廓为茶棕色细虚线。
- 眉头、眉峰、眉尾只用小圆点标记，不显示文字标签。
- 脸部/五官定位线使用浅灰棕低透明线条。
- 脸部/五官定位线始终显示，帮助用户持续对齐。
- 所有辅助线仅用于屏幕参考，不参与最终保存照片。
- 用户可通过透明度滑杆调整辅助线整体可见度。

## 8. 眉形模板与智能推荐

### 8.1 V1 眉形模板

- 自然眉
- 标准眉
- 平眉
- 弯月眉

### 8.2 底部眉形卡片栏

- 识别成功后默认显示眉形卡片栏。
- 第一张卡片为「推荐：某眉形」。
- 推荐卡片内展示推荐理由。
- 每张卡片展示眉形线稿预览和眉形名称。
- 当前选中卡片高亮。
- 用户点击卡片后，上方眉形辅助线立即切换。

### 8.3 智能推荐规则

- V1 使用规则算法 + 端侧识别能力。
- 不接入大模型。
- 不上传用户照片做云端 AI 分析。
- 根据脸型、五官比例、眼距、眉眼距离等规则推荐眉形。
- 推荐文案保持温和，不评价用户缺陷。

### 8.4 推荐文案示例

- 你的脸型偏圆，推荐标准眉，略带眉峰更显立体。
- 你的脸型偏长，推荐平眉，有助于平衡脸部比例。
- 你的五官比例较柔和，推荐自然眉，更适合日常妆容。

## 9. 微调需求

### 9.1 入口

- 辅助线生成后，底部操作区提供「微调」按钮。
- 点击后打开底部调节面板。
- 默认双眉联动调整，不支持左右眉单独调整。

### 9.2 滑杆项

- 上下移动
- 左右移动
- 大小缩放
- 旋转角度
- 透明度

### 9.3 建议范围

- 上下：-30px 到 +30px。
- 左右：-30px 到 +30px。
- 大小：80% 到 120%。
- 旋转：-10° 到 +10°。
- 透明度：20% 到 100%。

### 9.4 操作规则

- 调节时辅助线实时变化。
- 提供「重置」按钮，恢复系统生成状态。
- 提供「完成」按钮，收起微调面板。
- 重新定标后不保留上一次眉形模板和微调参数。

## 10. 画眉模式

### 10.1 显示方式

- 双眉同时显示。
- 不做左眉/右眉分步模式。
- 不提供语音引导。
- 不提供画眉过程文字步骤。
- 仅保留必要状态提示。

### 10.2 画眉页底部区域

- 眉形卡片横滑栏。
- 操作按钮区：
  - 完成并保存
  - 微调
  - 重新定标

### 10.3 重新定标

- 用户点击「重新定标」后重新识别人脸。
- 识别成功后重新生成辅助线和推荐眉形。
- 清空之前模板选择和微调参数。

## 11. 保存照片需求

### 11.1 保存规则

- 用户点击「完成并保存」。
- 系统保存当前真实相机画面。
- 最终照片不显示辅助线。
- 保存成功后进入保存成功页。

### 11.2 相册权限失败

- 弹出权限引导弹窗。
- 文案说明：需要相册权限才能保存画眉效果图。
- 按钮：
  - 去开启权限
  - 暂不保存

### 11.3 保存成功页

- 不展示照片预览。
- 不提供分享按钮。
- 展示标题：保存成功。
- 展示说明：画眉效果图已保存到相册。
- 主按钮：重新画眉。
- 次按钮：返回首页。

## 12. 权限需求

### 12.1 相机权限

- 用户进入相机页时请求相机权限。
- 如果拒绝或未授权，展示权限引导页。
- 页面说明：MS 智能画眉需要使用相机来显示实时画眉辅助线。
- 主按钮：去开启权限。
- 次按钮：返回首页。

### 12.2 相册权限

- 仅在用户保存照片时请求。
- 授权失败时展示权限引导弹窗。

### 12.3 微信资料权限

- 登录时获取昵称、头像。
- 不获取手机号。

## 13. 账号与云开发

V1 使用微信云开发。

### 13.1 用途

- 微信登录。
- 用户身份初始化。
- 用户基础资料存储。
- 后续会员/付费权益预留。

### 13.2 用户表建议字段

```js
{
  openid: string,
  nickname: string,
  avatarUrl: string,
  firstLoginAt: Date,
  lastLoginAt: Date,
  isMember: false,
  memberLevel: "free",
  createdAt: Date,
  updatedAt: Date
}
```

### 13.3 V1 不保存

- 用户照片。
- 人脸关键点数据。
- 画眉过程图。
- 手机号。
- 历史效果图。

## 14. 个人中心

### 14.1 入口

- 首页右上角「我的」。
- 未登录时先登录，登录成功后进入个人中心。

### 14.2 页面内容

- 用户头像。
- 用户昵称。
- 会员状态占位：普通用户。
- 入口顺序：
  - 隐私说明
  - 关于我们
  - 联系客服/意见反馈

### 14.3 不提供

- 退出登录。
- 付费入口。
- 历史照片。
- 历史记录。

### 14.4 联系客服/意见反馈

- V1 接入微信客服能力。
- 点击后打开微信客服会话。

## 15. 隐私说明页

隐私说明做成独立页面。

页面内容包括：

- 相机权限仅用于实时画眉辅助。
- 相册权限仅用于保存用户主动拍摄的画眉效果图。
- 昵称头像用于账号展示和用户身份识别。
- V1 不上传、不保存用户照片。
- V1 不保存人脸关键点数据。
- 用户可在微信设置中管理小程序权限。
- 联系方式/主体信息占位。

## 16. 关于我们页

页面内容包括：

- 产品名：MS 智能画眉。
- 副标题：AI 相机画眉助手。
- 产品介绍：帮助用户通过相机辅助线找到眉头、眉峰、眉尾，提升画眉效率。
- 版本号：V1.0.0。
- 联系方式占位。
- 返回个人中心入口。

## 17. V1 不做范围

V1 暂不做：

- 实时人脸持续追踪。
- 辅助线实时跟随脸部移动。
- 大模型 AI 分析。
- 云端上传照片分析。
- 上传照片生成静态参考图。
- 语音引导。
- 左右眉单独画眉模式。
- 左右眉分别微调。
- 付费入口。
- 会员购买。
- 模板商城。
- 正式埋点分析。
- 历史照片/效果记录。
- 分享功能。
- 后台模板配置。

## 18. 技术可行性建议

V1 推荐技术方案：

- 微信小程序原生开发。
- 微信云开发承载登录和用户表。
- 相机使用小程序 camera 能力。
- 定标时取当前相机帧做人脸识别。
- 端侧做人脸关键点/脸型比例判断。
- 根据规则算法生成推荐眉形。
- 使用 Canvas 或覆盖层绘制固定辅助线。
- 画眉过程中不持续识别，降低性能风险。
- 保存时拍摄净图，不合成辅助线。

## 19. 验收标准

- 用户可从首页登录后进入相机页。
- 用户可授权相机并看到前置相机画面。
- 用户点击开始定标后出现识别状态。
- 单人正脸、光线正常时可生成辅助线。
- 生成内容包含脸部定位线、五官定位线、眉形轮廓线、关键点圆点。
- 系统可推荐一个眉形并展示推荐理由。
- 用户可切换 4 种眉形模板。
- 用户可通过滑杆微调辅助线。
- 用户可重新定标，且状态重置。
- 用户可保存不带辅助线的净图。
- 保存成功后进入保存成功页。
- 用户可从首页进入个人中心。
- 个人中心可进入隐私说明、关于我们、联系客服。
